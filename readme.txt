---- Guia para utilização do Projeto ----

- Para abrir o projeto, abra o arquivo index.html no navegador
- A primeira seção é a de adicionar os campos de texto, que possuem as validações que foram solicitadas.
- A segunda seção é a de adicionar os botões, que tem as cores do texto e do botão personalizáveis, com as valdações solicitadas de poder conter apenas 3 botões na tela.
- A terceira seção é a de seleção de imagem de fundo do formulário, e quando é clicado em alguma imagem, a mesma se torna o plano de fundo do formulário.

Observações:
Ao adicionar um botão de submit do formulário e clicar no mesmo, são exibidos os valores dos campos que foram adicionados e preenchidos em uma seção chamada Valores dos Campos do Formulário, que aparecerá assim que o submit for aplicado, logo abaixo do formulário gerado.
